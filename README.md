# Installation

#### Update composer
```
composer update
```

```
cp .env.example .env
```

#### Migrate Databases
```
php  artisan migrate --seed
```

#### Generate Key
```
php artisan key:generate
```
#### Start server
```
php artisan serve
```

#### Account

###### Super Admin
```
us: admin
pw: Admin123
```

###### Normal User
```
us: user
pw: User123
```

And u are ready to go!!
